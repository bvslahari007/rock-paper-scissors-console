# Console Rock Paper Scissors

A console-based Rock Paper Scissors game built using JavaScript.

This project was created to practise core JavaScript fundamentals such as:
- Functions
- Conditionals
- Loops
- Random number generation
- User input handling
- Game logic

The game runs entirely in the browser console and allows the user to play 5 rounds against the computer.

---

## Features
- Random computer moves
- User input using prompt()
- Score tracking system
- 5-round gameplay
- Final winner announcement

---

## Tech Used
- HTML
- JavaScript

---

## What I Learned
While building this project, I practised:
- Breaking problems into smaller functions
- Debugging logic errors
- Working with randomness using Math.random()
- Comparing conditions effectively
- Structuring a simple game loop

---

## How to Run
1. Clone the repository
2. Open `index.html` in your browser
3. Open the browser console
4. Play the game through prompts and console output
